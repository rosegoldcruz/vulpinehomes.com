# 3D Models Directory

## Required File

Place your `vr-foxy.glb` file in this directory.

The AR Fox component expects the model at: `/models/vr-foxy.glb`

## Model Requirements

- Format: GLB (binary glTF)
- Recommended size: < 10MB for optimal web performance
- Should include animations (e.g., "idle" animation)
- Optimized for mobile AR viewing

## Testing

You can test the AR fox at: `/ar-fox-test`
